# TR Technologies Group Website

A minimal, professional corporate website for TR Technologies Group built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Multi-page responsive website (Home, Services, Projects, About, Contact)
- Monochrome grayscale design system
- Custom typography using Mona Sans and Hubot Sans
- Optimized for performance and accessibility
- Edge-to-edge hero banner with Unsplash imagery
- Mobile-responsive navigation

## Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Color Palette

- Primary Headings / Navigation: `#111111`
- Body Text: `#2B2B2B`
- Secondary / Muted Text: `#6B6B6B`
- Icon / Accent Grey: `#8D8D8D`
- Borders / Dividers: `#D2D2D2`
- Card / Panel Backgrounds: `#E6E6E6`
- Page Background: `#F5F5F5`
- Disabled / Low Emphasis: `#F0F0F0`

## Typography

- **Headings**: Mona Sans (weight 600–700)
- **Body Text**: Hubot Sans (weight 400–500)
